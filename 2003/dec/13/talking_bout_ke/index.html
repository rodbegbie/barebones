Talking ‘bout Key Generation

The latest LiveUpdate of Norton AntiVirus 2004 will helpfully search your hard drive and report if you happen to have… a KeyGen for pirately installing Norton AntiVirus 2004. Isn’t that helpful…

(Note: My copy of NAV is legit. I just happen to have a KeyGen for “research purposes”.)

2 comments | ✲

Comments

On Sunday, December 21, 2003, A User commented:

Yes, I've gotten the same report over the past few weekly scans. But when you think about it, they're doing you a favor. Some viciously clever viruses do, in fact, patch and disabl NAV and other security programs. Thus, detecting that you have a program that also locates and analyzes a portion of NAV (or its Registry fields) is a reasonable thing to do. Note that the scanner does not alter the file in question, nor does it notify Symantec; it merely flags it for your own decision-making.

On Saturday, January 3, 2004, Another User commented:

This is all very true, but however, Norton doesn't have the ability to work out which software will be able to change registry settings for Norton, because to do that it would have to run every program on your machine and work out what it is changing, which is something it doesn't do. It is simply a ploy to scare people who don't do much pirating into buying the software. If you didn't know what I meant, wouldn't you start telling all your friends not to install the pirate version of Norton Antivirus because it has a virus built into the crack?

This is what they call Marketing in a strange and twisted kind of way.
