A Cunning Web

9 months ago, Danny O’Brien suggested that the best way to stress-test BitTorrent would be to make the newly-released Debian CDs available via it.

This afternoon, the brand-spanking new RedHat 9 was released. One quick slashdotting later, and a massive network of people helped serve the three ISO files, offering download speeds reported to be faster than those offered by RedHat’s own servers.

Indeed, I managed to get all 1.72Gb within four hours, with top speeds often coming close to the 1.5Mbps of my DSL line. Hugely impressive.

RSI Reduction

Today’s NTK linked to a project called WorkRave, a tool to help keybard-jockeys such as myself avoid RSI by running timers telling you to take a break.

It has many kewl features, not least the fact that it networks across multiple PCs, so it can track when I’m spending time on my Linux box instead of my main PC.

One thing it is lacking, though, is Mac OSX support. The backend code should run OK, but it’d need a front-end. Perhaps a Konfabulator widget? (Summoning the awesome power of the LazyWeb. Come in LazyWeb)
