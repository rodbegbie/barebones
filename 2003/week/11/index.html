➠ March 21, 2003

RSI Reduction

Today’s NTK linked to a project called WorkRave, a tool to help keybard-jockeys such as myself avoid RSI by running timers telling you to take a break.

It has many kewl features, not least the fact that it networks across multiple PCs, so it can track when I’m spending time on my Linux box instead of my main PC.

One thing it is lacking, though, is Mac OSX support. The backend code should run OK, but it’d need a front-end. Perhaps a Konfabulator widget? (Summoning the awesome power of the LazyWeb. Come in LazyWeb)
