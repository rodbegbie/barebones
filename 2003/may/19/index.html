Syndirella — Now with added Toast

Syndirella is in a rather unpleasant state of limbo at the moment. The original author’s stated that he pretty much wants to give up development, and has been reasonably unreachable of late.

So here’s a release of a change I’ve wanted for a while. This release include pop-up “toast”, as seen in apps such as MSN Messenger or DeadAIM, alerting you to new stories.

These were implemented using code from The Genghis Project.

Known issues/TODOs:

  • Only one piece of toast can display at a time (though this may be a feature, since if 20 blogs updated, you’d fill your screen pretty quickly)
  • There’s no option to turn it off. There is now!
  • You can’t click on a piece of toast and restore the main window. You can now!
  • The copyright statement for the Genghis Project hasn’t been added anywhere. Consider this part of the documentation: Portions copyright 2002–2003 The Genghis Group (http://www.genghisgroup.com/).
