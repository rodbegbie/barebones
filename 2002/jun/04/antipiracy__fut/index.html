Antipiracy == Futility

➠ June 4, 2002

Antipiracy == Futility

Yesterday, Microsoft released a Service Release for Office X, their Mac OS X office suite.

In addition to the promised “1,000 performance improvements, updates, and fixes”, was a nasty sting in the tail for naughty people. Microsoft has disabled the CD Keys (required for installation) that were in common circulation.

So, did the pirate community rush out to CompUSA and purchase legal copies forthwith? Probably not. Within 4 hours of the SR’s release, a new serial was posted which works like a charm.

It halted the illegal users almost as much as the “Only one person can use a CD Key at a time on the local network” rule which can be overridden using OS X’s built-in firewall.

The reason I bring this up is because MS has already announced that Windows XP Service Pack 1 will disable the key most pirate copies are using (see end of the article). Anyone want to place bets as to how long (in minutes) after SP1’s release, a new code will be out and about?
