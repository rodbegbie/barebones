➠ December 18, 2002

begbie.com Comes Home

I’ve finally got around to copying my begbie.com website over onto the same machine that hosts the rest of the sites I manage.

It’s quite touching. I’ve hosted it at various ISPs over the last few years, but now, for the first time since it was located on the Heriot Watt CEE Dept network (with the catchy URL http://www.cee.hw.ac.uk/~ceerab/), I am able to run ‘tail -f’ on the access_logs and watch the hits as and when they occur. It’s really rather touching that The Love Calculator—a shoddy bit of Perl code that I knocked up with paper and pen in my student flat, then typed into a Uni machine the next day—is still as popular today with pre-teens and AOL users (not necessarily distinct groups) as it ever was.
