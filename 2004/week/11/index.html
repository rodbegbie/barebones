➠ March 17, 2004

Excel-lent

When you’re holding Excel, everything looks like a spreadsheet—A great post by Yoz comparing Excel to the Unix command line.

I absolutely agree. In my programming career, I’ve seen some incredibly whacked-out uses for Excel: Loading test data for systems, creating vast sprawling directory structures for reports, and (my personal favourite) a giant matrix defining screen-level security for an application, reading in Java code and spitting out SQL.

Sure you could use shell scripts or Perl to do all these things, but it would be considerably less pretty and far scarier to the non-Unix set.

I love Excel dearly. It’s by far the best thing Microsoft have ever done.
