PartitionMagic Kills Hard Drives

Far too much of my weekend was lost to my Windows PC, which completely died after I attempted to resize the partitions using PartitionMagic. Sadly, this is not the first time that PartitionMagic has fried a hard drive of mine. Fool me once, etc.

This was a real beauty of a crash too. It managed to scramble the hard drive so skillfully that every Windows installation CD I tried refused to start up while the hard drive was still connected! Thank gawd for my handy business card Linux CD which saved the day. Best $5(EFF Store: GNU-Linux Bootable Business Cards (BBC’s)) I ever spent. A quick parted(Parted – GNU Project – Free Software Foundation (FSF)) later, and I was off.

Best of all, my backup software (installed in a sleep deprived state) actually worked! Restoration was quick and painless.

And now, for my own reference next time I’m reinstalling Windows, here’s my current Must Install list—The very basics that I need, in order to have a fully functioning Windows desktop:

ReGet Deluxe, Cygwin, DAEMON Tools, WinZip, WinRAR, Firebird, Thunderbird, vim, TweakUI, PuTTY, FileZilla, RealVNC
